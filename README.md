## 🛠️ Technology Stack
This project is built using modern, high-performance web technologies:

- **Frontend Framework:** React 19 + Vite
- **Styling:** Tailwind CSS (Custom Dark Theme & Glassmorphism)
- **Animations:** Framer Motion (Scroll triggers, parallax, stagger effects)
- **Background Effects:** `react-tsparticles` (Space/Galaxy simulation)
- **Icons:** `lucide-react`
- **Navigation:** `react-scroll` (Smooth anchor routing)

## ✨ Key Cinematic Features

1. **Interactive Galaxy Background:** A dynamic, slow-moving starfield generated by `tsparticles` that reacts to window resizing.
2. **Magnetic Navigation Pills:** A floating glassmorphism navbar where links magnetically pull toward the custom cursor.
3. **Scroll-Triggered Story Timeline:** A vertically animated journey timeline that draws a gradient line and fades sections in as the user scrolls.
4. **3D Glass Project Cards:** Project showcases feature `perspective-1000` 3D rotation, dynamic gradient backdrops, and hover translation.
5. **Custom Cursor:** A custom-built dot-and-ring cursor system that replaces the default browser pointer, enhancing the immersive feel.
6. **Staggered Typography:** The hero section features letter-by-letter stagger reveal animations for a premium movie-like introduction.

## 📦 Local Installation & Setup

To run this project locally on your machine, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Medha207/my-portfolio.git
   cd my-portfolio
   ```

2. **Install dependencies:**
   *(Ensure you have Node.js installed)*
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **View the application:**
   Open your browser and navigate to exactly:
   [http://localhost:5173/](http://localhost:5173/)
   
   *(Note: If you encounter a "Connection Refused", try using `http://127.0.0.1:5173/` or run the server with `npm run dev -- --host`)*

## 📂 Project Structure

- `/src/components` - All React UI modules (Navbar, Hero, Projects, etc.)
- `/src/data/portfolioData.js` - The central data store. Edit this single file to update all text, skills, and projects globally across the site without touching UI code.
- `/src/index.css` - Global Tailwind directives and custom animation keyframes (`@layer utilities`, `glass-panel`, `.text-gradient`).
